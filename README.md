# youtubeclone
📺 YouTube Clone (React)

A responsive YouTube-inspired web application built with React, React Router, and RapidAPI.

🚀 Features
	•	Dynamic category routing (/category/:category)
🛠 Tech Stack
	•	React
	•	React Router DOM
	•	Axios
	•	RapidAPI (ytjar)
	•	TailwindCSS
	•	React Icons

 # Resources
- API: https://rapidapi.com/ytjar/api/yt-api

📦 Installation

git clone https://github.com/secilvergili/youtubeclone.git
cd youtubeclone
npm install
npm run dev

🔑 Environment

Create a .env file:
VITE_RAPIDAPI_KEY=your_api_key
